# Optio W90 — Field Gallery

Photos and videos from my Pentax Optio W90 — a rugged, waterproof field camera.

Powered by [Charmera](https://github.com/timncox/charmera) — a macOS menu bar app
that imports straight off the camera card and publishes here automatically.

The gallery aesthetic mirrors the W90 itself: armored black body, pistachio-lime
accents, brass hex fasteners, and instrument-panel readouts.
